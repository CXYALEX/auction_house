## Usage
Before run this project, the foundry(0.2.0) is required to be installed.
Make sure you have install the foundry(0.2.0) 
```
forge --version
forge 0.2.0 (5c3b075 2024-03-08T00:16:55.254956000Z)
```
### install dependency
```shell
npm install 
```

### Anvil

```shell
$ make anvil
```

### Deploy

```shell
$ make deploy
```

### Build (if require)

```shell
$ make build 
```

### Test (if require)

```shell
$ make test
```
